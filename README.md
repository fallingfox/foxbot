# FoxBot
## Textový Bot pro FallingFox komunitní Discord server

### Technologie
* programovací jazyk JavaScript
* backend JavaScript engine `node.js`
* správce balíčků `npm`
* knihovna **dotEnv**
`npm install dotenv`
* knihovna `discord.js`
